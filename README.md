# Job-Portal Website

<li>Clone the repository 

`git clone https://github.com/preethasarkar/E-Cell-Task`

`cd E-Cell-Task`
</li>
<li>Install dependencies

`npm install`

</li>
<li>Run the project

`npm start`
</li>
